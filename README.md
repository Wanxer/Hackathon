# Damm Smart Planner

**Damm Smart Planner** és una eina per optimitzar una ruta real de repartiment de Damm combinant optimització de ruta, geocodificació, comparativa amb la ruta original i planificació de càrrega del camió.

El projecte treballa sobre la ruta real:

- **Ruta:** DR0051
- **Data:** 2026-02-05
- **Transport:** 11443257
- **Repartidor:** 855190
- **Clients:** 24
- **Depot:** DDI Mollet

## Objectiu del projecte

L’objectiu és reduir quilòmetres de repartiment i facilitar la feina del repartidor amb una càrrega del camió més ordenada i explicable.

El projecte resol quatre blocs:

1. Preparació de dades i baseline real.
2. Optimització de la ruta.
3. Optimització de la càrrega del camió.
4. Dashboard final i informes PDF.
